## Scripts no Projeto

Os scripts disponíveis para a execução do projeto são:

### `npm start`

Roda o app em modo de desenvolvimento.\
Site disponível em [http://localhost:3000](http://localhost:3000)

### `npm run build`

Constrói a aplicação para produção na pasta `build`.\
